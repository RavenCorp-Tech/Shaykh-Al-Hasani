# Shaykh Al-Hasani - Islamic Scholarship Website

A beautiful, modern, fully responsive static website showcasing the translated works and teachings of **Shaykh al-Muḥaddith Abū ʿAlī al-Ḥārith bin ʿAlī al-Ḥasanī**, a renowned hadith scholar from Iraq.

## 🌟 Features

### Design & UX
- **Modern UI Design**: Clean, professional interface with sophisticated styling
- **Dark/Light Theme**: Fully functional theme toggle with persistent storage
- **Smooth Animations**: Reveal effects, hover effects, transitions, and scroll animations
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Beautiful Typography**: Proper fonts with excellent readability
- **Professional Color Scheme**: Primary green (#1a472a) with accent gold (#d4a574)

### Functionality
- **Navigation**: Sticky header with smooth scrolling and mobile menu
- **Theme Toggle**: Dark/Light mode with system preference detection
- **Scroll Reveal**: Elements animate into view as user scrolls
- **Article System**: Full article management with footnote system
- **Footnote Navigation**: Clickable footnote references that scroll to content
- **Mobile Optimization**: Touch-friendly interface with responsive layout

### Content Pages
1. **Home (index.html)**: Hero section with resource overview
2. **Articles (articles.html)**: Scholarly articles with metadata
3. **Article Detail (article-detail.html)**: Full article with footnotes
4. **Fatāwā Ḥadīthiyyah**: Islamic rulings
5. **Books**: Published works with translations
6. **Videos**: Video lectures
7. **Images**: Educational graphics
8. **Ḥadīth**: Collections
9. **Encyclopedia**: Narrator biographies
